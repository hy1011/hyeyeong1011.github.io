title: (CSS)display-none과 visibility-hidden의 차이
---

<br>

display: none과 visibility: hidden 둘 다 요소를 숨겨주는 기능을 한다.   
하지만 차이점이 있다.

<br>

1.visibility: hidden

``` css

.box1{
  visibility: hidden;
}

```
<br>

<br>
visibility는 요소는 사라지지만 눈에만 안보일 뿐 영역은 그대로 남아있다. 

<br>
<br>

2.display: none

``` css

.box1{
  display: none;
}

```
<br>

<br>
display: none은 요소가 사라지며 영역도 차지하지 않는다.
